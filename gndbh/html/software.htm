In the directory "Software" of this CD you'll find some useful software for NDB DXers:

  • GCMWIN to create great circle maps
  • QRBQTE to calculate distances and bearings
  • SPECTRAN and ARGO to visualize and analyze NDB using your computer's soundcard
  • Brian Keyte's B_KEYER if you'd like to "crack" negative keying NDBs

Please copy the contents of the respective directory into a directory of your choice on your PC.
